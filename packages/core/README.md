# @prompt-registry/core

Core domain types and interfaces for Prompt Registry.

This package contains the fundamental domain models, ports, and public interfaces used throughout the Prompt Registry ecosystem.

## Installation

```bash
pnpm add @prompt-registry/core
```

## Usage

```typescript
import { BundleManifest, SourceId } from '@prompt-registry/core';
```

## Structure

- `domain/` - Core domain types and models
- `ports/` - Interface definitions for external dependencies
- `public/` - Public API exports
